# AI-Powered Cyber Incident Monitoring System

A backend system designed to detect, monitor, and analyze cybersecurity incidents in real time.

This project simulates a **Security Operations Center (SOC)** backend.

## Features

- JWT Authentication
- Role-Based Access Control
- Incident Management
- AI Threat Classification
- Brute Force Login Detection
- Incident Spike Detection
- Real-Time Security Alerts (WebSocket)
- Security Analytics APIs
- Audit Logging

## Tech Stack

Backend:
- Java
- Spring Boot
- Spring Security
- MongoDB
- WebSocket

AI:
- Spring AI
- OpenAI API
